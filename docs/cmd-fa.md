# Form Action Usage
## Command `fa`
```
[:] $ fm fa -h     
Form Action

Usage: fm fa [arguments] [options]

Arguments:
  ControlID      Control Id
  Command        Command
  Value          Value

Options:
  -i|--index     Add or Remove Zero-based Index
  -rm|--remove   Remove Form Action
  -of|--outFile  Write Result to File
  -?|-h|--help   Show help information
```
